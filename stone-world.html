Featured Project

Stone World

An RPG game inspired by Dr Stone, featuring complex gameplay mechanics, player progression systems, and an immersive player-driven economy. Built with a dedicated team of developers working collaboratively in an integrated development ecosystem.

Active FullStack Development Roblox Platform

Project Overview

Stone World is a sophisticated game ecosystem where I contribute as a FullStack Developer. The project encompasses complex system architecture, data management, and user-facing gameplay mechanics. I've been instrumental in designing and implementing several core systems that define the player experience.

My Role & Team Structure

Head Developer for Scripters

As the Head Developer, I oversee all scripting team members and ensure code quality, architecture consistency, and proper system integration across all frameworks. My responsibility is to guide, mentor, and coordinate the scripter team to deliver robust and scalable solutions.

  • Leadership of scripting team
  • Code review and quality assurance
  • Architecture design and documentation

Team Composition

The Stone World project is powered by a dedicated team of approximately 14 members, consisting of experienced scripters, talented builders, and specialized developers who work in perfect harmony. Each team member brings unique expertise and contributes to different aspects of the game ecosystem.

Lead Scripter & Discussion Groups

Each scripter has their dedicated discussion group where we collaborate on specific systems and areas of expertise. These groups facilitate real-time communication, code reviews, and architectural discussions. As the Lead, I actively participate in all groups to provide guidance, direction, and ensure alignment with the overall game vision.

Collaboration Benefits:

  • Real-time code collaboration
  • Quick problem resolution
  • Knowledge sharing
  • System integration oversight

Integrated Development Structure

The game architecture follows a centralized development approach where all developers work in unified spaces and integrate their work across all game places. This structure ensures consistency, reduces conflicts, and allows for seamless system communication across the entire game world.

My Responsibility:

I coordinate the entire development workflow, manage system integration, facilitate communication between teams, and ensure all developed systems work harmoniously together across Testing Place, Main Menu, and all game maps.

Custom Framework Architecture

I've designed and implemented a custom framework that serves as the backbone of Stone World's architecture. This framework provides standardized patterns, communication protocols, and integration points that all systems build upon. It enables scalability, consistency, and efficient collaboration across the entire development team.

Framework Features:

  • ✓ Modular architecture
  • ✓ Service-oriented design
  • ✓ Cross-system communication
  • ✓ Data synchronization layer
  • ✓ Error handling & logging
